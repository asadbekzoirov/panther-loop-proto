The status pill for a task's lifecycle state. Each status maps to a fixed bg/fg color pair.

```jsx
<StatusBadge status="open" />
<StatusBadge status="in-progress" />
<StatusBadge status="confirmed" />
```

Statuses: `open`, `in-progress`, `confirmed`, `disputed`, `cancelled`, `expired`. 11pt/600 text, fully rounded pill, 22pt tall.
