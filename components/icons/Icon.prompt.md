Monochrome Lucide icon that inherits color via CSS mask — the PantherLoop icon system.

```jsx
<Icon name="home" size={24} color="var(--navy)" />
<Icon name="plus" size={20} color="#fff" />
```

Use Lucide names (kebab-case). Default size 24, color `currentColor`. Tint by setting `color`. Common icons: home, plus, message-circle, bell, user, map-pin, star, clock, camera, chevron-right, search, shield-check.
